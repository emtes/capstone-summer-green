# Usage

<!-- usage -->

```sh-session
$ npm install -g summergreen
$ summergreen COMMAND
running command...
$ summergreen (-v|--version|version)
summergreen/0.0.0 linux-x64 node-v14.6.0
$ summergreen --help [COMMAND]
USAGE
  $ summergreen COMMAND
...
```
